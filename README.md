# Retail Sales Data Analysis

## Project Overview
This project analyzes retail sales data using Python, Pandas, NumPy, Matplotlib, and Seaborn.

## Objectives
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Sales Trend Analysis
- Customer Insights
- Data Visualization

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Retail Sales Dataset

## Key Insights
- Identified top-performing products
- Analyzed sales trends
- Visualized customer purchasing behavior
- Generated business insights from data
